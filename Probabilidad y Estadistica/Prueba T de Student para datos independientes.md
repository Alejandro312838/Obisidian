La prueba t de Student para datos independientes es una ==prueba estadística utilizada para comparar las medias de dos grupos y determinar si existe una diferencia significativa entre ellas== Se utiliza comúnmente para comparar si las medias de dos poblaciones son diferentes, especialmente cuando el tamaño de la muestra es pequeño o la desviación estándar de la población es desconocida.
